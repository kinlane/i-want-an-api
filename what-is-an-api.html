What Is An API?

An API, or Application Programming Interface, at its most basic level, allows your product or service to talk to other products or services. In this way, an API allows you to open up data and functionality to other developers and to other businesses. It is increasingly the way in which companies exchange data and services, both internally and externally.

An API works just like web pages on the Internet, but instead of being for humans, it is mean for other machines, web and mobile applications to use.
